# deploy/docker

Специфичные для окружения Compose-оверрайды (`docker-compose.staging.yml`,
`docker-compose.prod.yml`) и вспомогательные entrypoint-скрипты
контейнеров, добавленные на этапе DevOps (`docs/spec/08-devops.md`). Базового
`docker-compose.yml` в корне репозитория достаточно для локальной
разработки.
